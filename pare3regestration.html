Home runs + strikeouts" suitable for two-sword style evaluation in a true sense

Until now, Ohtani's dual-wielding style has been compared to Babe Ruth, and "double-digit wins and double-digit homeruns" have been used as indicators. And he has achieved the best feat since Ruth last season. However, personally, unlike home runs, which can be accumulated by individual power, the number of wins is greatly influenced by the team, so I felt that it was an unsuitable indicator for purely evaluating two-way fighting. For a pitcher, the closest thing to a home run is strikeouts, and "the number of home runs + the number of strikeouts" is probably the most appropriate indicator for evaluating a two-sword style.
